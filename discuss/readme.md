
## 目录说明

本目录用于存放方案讨论记录。

讨论记录需要在相关 commit 前归档，文件名按照时间+主题命名。

单个讨论记录文件内部按照时间分段记录，每个时间段使用独立标题；最新时间段放在最上方。

每个时间段优先采用对话形式记录，保留用户输入原话，再记录助手回复中的讨论结论、设计判断和待确认问题。

不要默认记录每次对话；仅当用户明确要求记录、归档或写入 discuss 时，才更新本目录下的讨论记录。

## 历史归档文件


历史归档文件只作为已完成阶段的讨论记录和设计依据。除非用户明确指定某个历史归档文件，否则不要把新的对话内容继续归档到历史归档文件中。
